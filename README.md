# radar_data_classifier
This module attempts to classifcy the data files into radar data soruce.

## uv install
The 'uv' is the package manager that is used in this code for managing dependencies and executing the script.
If uv isn't installed in the system. Please do so by running the following commnand (Assuming it is Linux machine).

- curl -LsSf https://astral.sh/uv/install.sh | less


## Virtual env setup
- uv sync
This command will create the venv using the pyproject.toml present in the directory.

## Run the script
Activate the venv and run the code using,
- python main.py
